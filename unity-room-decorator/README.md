# MoonRoom Unity Project

A cozy low-poly room decorator sandbox built with Unity 2022.3 LTS.

## Features

- **Placement System**: Click to select items, place them in the room with mouse
- **Rotation**: Press R to rotate items 90 degrees
- **Save/Load**: Persist your room layout to JSON
- **Catalog System**: Easily add new placeable items

## Getting Started

1. Open Unity Hub
2. Click "Add" and select this folder
3. Open with Unity 2022.3 LTS
4. Open `Assets/_Project/Scenes/Room_V0.unity`
5. Press Play
